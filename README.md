# 🎯 Kanban de Prospecção — Academias / CrossFit / Personal

Funil de prospecção B2B conectado ao Supabase. Leads extraídos da base de
Dados Abertos da Receita Federal (CNAE 9313-1/00 — atividades de condicionamento físico).

## Uso
Abra `index.html` (ou via GitHub Pages). Arraste cards entre as colunas do funil.
Cada card tem botão direto de WhatsApp com mensagem pronta.

## Stack
- HTML/CSS/JS puro (zero build)
- Supabase (PostgREST) como base central de leads
